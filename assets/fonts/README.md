# Fonts Directory

This directory contains font files used for generating Shadowverse dashboard images.

## Required Fonts

- **NotoSans-Regular.ttf**
- **NotoSans-Bold.ttf**

## Download Instructions

1. Visit: https://fonts.google.com/noto/specimen/Noto+Sans
2. Click the "Download family" button
3. Extract the downloaded ZIP file
4. Copy `NotoSans-Regular.ttf` and `NotoSans-Bold.ttf` to this directory

## Alternative: Direct Download

You can also download directly from Google Fonts:
https://fonts.google.com/download?family=Noto%20Sans

## Fallback

If fonts are not found, the system will automatically fall back to PIL's default font with a warning message.
